# Digitalgardn Frontend

Interface utilisateur de l'application Digitalgardn. Ce projet est initialise avec Vite et utilisera un framework JavaScript moderne (React).

---

## Technologies utilisees
- **React 19**
- **Node.js v22.17.1**
- **npm 10.9.2**

---

## Installation et Lancement

Suivez ces etapes pour configurer et lancer le projet sur votre machine locale.

**1. Cloner le projet**

```bash
git clone https://github.com/VOTRE_NOM/digitalgardn-frontend.git
cd digitalgardn-frontend
npm install
npm run dev
```
